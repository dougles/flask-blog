# Flask blog  tutorial 

### Create virtual environmnet

 `python3 -m venv venv`

### Activate  virtual env

`source venv/bin/activate`

### Execute application 

`export FLASK_APP=flaskr`

`export FLASK_ENV=development`

`flask init-db`

`flask run`


## Run Tests

`pip install pytest coverage`

## Unit Tests

`pytest`

## Test coverage

`coverage run -m pytest`
